# Pins

 ------------------------
  Arduino        ST7032
 ------------------------
  3.3V    --+-- VDD
            +-- RST
  A4(SDA) --*-- SDA
  A5(SCL) --*-- SCL
  GND     ----- GND
  
  *... 10Kohm pull-up
 ------------------------
